# Global Stock Market Data Analysis

# Objective
Perform data cleaning, EDA, and visualization on global stock market indicators.

# Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

# Steps Performed
- Data loading and exploration
- Missing value handling
- Outlier treatment
- Feature engineering
- Exploratory Data Analysis
- Insight generation

# Key Insights
- Strong correlation between stock price variables
- Volatility varies significantly across countries
- Volume has limited impact on daily returns
